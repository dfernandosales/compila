# compila

primeiramentei, deve executar o comando "make clean; make;" para remover os arquivos gerados e gerar novos.

segundamente, após o make será gerado um executavel calc.exe. Para executar esse executavel deve-se passar um arquivo de teste como primeiro argumento.

            ex:    ./calc test/test.txt

no terminal será exibido, no "happy day", a arvore do .txt passado e após será exibido os print contidos no txt.





Fluxo:

    make clean;

    make;

    ./calc test/test.txt
